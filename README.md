# 撩鼻預約機

## User Guide
1. Download Python>=3.7 (64-bit)
2. Add python to your environment variable
3. Install library with cmd: 
    pip install selenium
    pip install json
4. Download source code: Code > Download ZIP
5. Edit config.json (Explanation below)
6. Run Main.py
7. Pray and wait
